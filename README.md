별도로 절차적 맵 생성과 FSM 으로 3D 방치형 게임 링크도 첨부했습니다.

## 🧙 사용법

1. 이 저장소를 클론하거나 PC 빌드를 다운받습니다,
2. 조작은 마우스로만 진행합니다.
3. 인벤토리에서 랜덤 아이템을 뽑아 장착합니다.
4. Coin을 얻습니다.
